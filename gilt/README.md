# GILT

*The machine age, gilded.* An **Art Deco** / machine-age design language — gilded geometry, jewel lacquer, and mirror symmetry, from the **1925 Paris Exposition** to the **Chrysler Building**.

- **`index.html`** — the documentation site (open it in a browser, or visit the GitHub Pages URL).

## Concept

The deliberate opposite of Memphis within the same impulse for ornament: symmetry instead of scatter, thin gold-and-black linework instead of cutout shadows, deep jewel lacquer instead of flat clash. Three color families — **Metals** (the gilding), **Jewels** (the lacquer), **Powders** (the boudoir) — plus a set of stylized motifs (sunburst, chevron, ziggurat, fan, fluting, fountain, inlaid frame).

## Type

Set with **Poiret One** (thin geometric deco display), **Jost** (Futura-lineage body), and **Pinyon Script** (engraved-invitation accent), loaded from Google Fonts; falls back to `system-ui`.
