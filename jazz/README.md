# JAZZ

*The 90s paper cup.* A pastel-postmodern design language built from the most 90s object alive — the disposable cup with the teal-and-purple **"Jazz" squiggle**.

- **`index.html`** — the documentation site (open it in a browser, or visit the GitHub Pages URL).

## Concept

Soft pastels, scattered confetti, rounded stickers, and the two crossing brushstrokes that started it all: teal under, grape over. Outline everything in ink, drop a hard blur-free shadow so each element peels off the page like a sticker, and round every corner. Three color families — **Signature** (the squiggle), **Pastels** (the confetti), **Pop** (turn it up) — plus a full component kit (buttons, cards, forms, tabs, badges, switch, sticker callout) and six motifs. A **Soft / Loud** theme toggle re-saturates the whole UI from one token change.

A cousin of *Memphis* and *Sonora* — same decade, softer voice.

## Type

Set with **Baloo 2** (chunky rounded display), **Quicksand** (soft geometric body), and **Shantell Sans** (hand-drawn marker accent), loaded from Google Fonts; falls back to `system-ui`.
