# CAIRN

A rugged, weatherproof wayfinding design language descended from **WPA national-park posters**, **USGS topographic maps**, and the **National Park Service Unigrid** system.

- **`index.html`** — the documentation site (open it in a browser, or visit the GitHub Pages URL).

## Concept

Color is terrain. A dark bark/soil/stone ground carries kraft-canvas text, biome wayfinding hues (pine, tundra, alpine, clay) that swap by region, and one high-vis "blaze" accent. Built to read in any light, like a trail sign.

## Type

Set with **Oswald** (condensed display / signage), **Asap** (body), and **Space Mono** (data / coordinates), loaded from Google Fonts; falls back to `system-ui`.

## License

A design study evoking public-domain federal design traditions (WPA, USGS, NPS Unigrid). Names are referenced descriptively.
