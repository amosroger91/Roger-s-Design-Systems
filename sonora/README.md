# SONORA

A loud, warm interface language of **neon dusk and Memphis confetti**, grounded in adobe and masa. Descended from **Taco Bell's 1990s rebrand**, when fast food went hot pink, purple, and a little bit jagged.

- **`index.html`** — the documentation site (open it in a browser, or visit the GitHub Pages URL).

## Concept

Warm Southwest ground — adobe, masa, desert night — struck by electric neon. *Run for the Border (1990) › the magenta bell (1995) › Yo Quiero (1997):* a mission-bell desert detonated into hot magenta, grape, and turquoise wrapped in Memphis-Group geometry — zigzags, confetti dots, jagged bursts — re-tokenized so you can design with it on purpose.

## Type

Set with **Bricolage Grotesque** (display), **DM Sans** (body), and **Space Mono** (data / accent), loaded from Google Fonts; falls back to `system-ui`.
