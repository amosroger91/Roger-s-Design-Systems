# AURORA

A glass interface language of translucent, light-bending chrome and flowing color — a mature, modern homage to **Windows Vista** and the **Aero** era of Frutiger optimism. *Edition 02.*

- **`index.html`** — the documentation site (open it in a browser, or visit the GitHub Pages URL).
- **`aurora.bak.html`** — a prior edition kept for reference.

## Concept

Frosted, layered glass over flowing aurora gradients: deep translucency, light-bending blur, and a calm, optimistic palette. Surfaces feel lit from within rather than printed on.

## Type

Set with **Open Sans** (UI / body) and **JetBrains Mono** (data / code), loaded from Google Fonts; falls back to `system-ui`.
